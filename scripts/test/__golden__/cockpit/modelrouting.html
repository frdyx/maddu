"Model Routing"

"Per-runtime + per-lane + per-pipeline modelPreference. Active hints by spawn."

"Per-runtime modelPreference" "GET /bridge/runtimes"
"◌"
"No runtime descriptors"
"Register a runtime under `.maddu/runtimes/` or via `maddu runtime add`."
"Per-lane defaults" "GET /bridge/lanes"
"◌"
"No lanes with model defaults"
"Add `defaults.modelPreference` to a lane in `.maddu/lanes/catalog.json`."
"Per-pipeline stage hints" "GET /bridge/pipelines"
"◌"
"No pipelines with stage hints"
"Pipeline stages can declare `modelPreference` to route different stages to different models."
